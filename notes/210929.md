### Git commit amend
```
git commit --amend
```
- 마지막 커밋 내용을 수정한다. (코드 또는 커밋 메시지 수정)
- 이전 커밋 이력은 삭제되고 해당 위치를 수정한 커밋 이력이 대신한다.
- 이미 push한 커밋 이력에 대해서 사용 시 주의해야 한다!!
